# Week 4 - NUnit & Moq

## 📖 Overview

This week introduces the fundamentals of **unit testing** and **mocking** in C#. The exercises demonstrate how to validate application logic using NUnit concepts and isolate dependencies using Moq concepts.

## 📂 Contents

### Task 1 - Standalone NUnit Lifecycle
- Test Setup
- Test Execution
- Assertion Validation
- Test Cleanup (TearDown)

### Task 2 - Standalone Moq
- Interface Mocking
- Dependency Injection
- Mock Object Creation
- Method Verification

## 🛠 Technologies Used

- C#
- .NET
- NUnit Concepts
- Moq Concepts

## ▶️ How to Run

Navigate to the required task folder and execute:

```bash
dotnet run
```

## 🎯 Learning Outcomes

- Understand the NUnit testing lifecycle.
- Learn the importance of unit testing.
- Understand dependency injection and mocking.
- Implement mock objects to isolate dependencies during testing.
