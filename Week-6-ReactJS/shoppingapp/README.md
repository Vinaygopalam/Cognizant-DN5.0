# ShoppingApp

A React class component application that displays an online shopping cart.

## Features

- Class components
- Props
- ReactDOM.render()
- Modern UI with table

## Tech Stack

- React 18
- Class Components
- CSS

## Installation

```bash
npm install
```

## Running the Application

```bash
npm start
```

The application will open at [http://localhost:3000](http://localhost:3000).
